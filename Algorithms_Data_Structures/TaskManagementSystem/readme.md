# Exercise 5: Task Management System

## 1. Understand Linked Lists

### Singly Linked List

A Singly Linked List consists of nodes where each node contains data and a reference to the next node. It allows forward traversal only.

### Doubly Linked List

A Doubly Linked List consists of nodes where each node contains data, a reference to the previous node, and a reference to the next node. It allows both forward and backward traversal.

## 2. Setup

A Task class is created with the following attributes:
- taskId
- taskName
- status

## 3. Implementation

A Singly Linked List is implemented to manage tasks.

The following operations are performed:
- Add Task
- Search Task
- Traverse Tasks
- Delete Task

## 4. Analysis

| Operation | Time Complexity |
|-----------|----------------|
| Add Task | O(n) |
| Search Task | O(n) |
| Traverse Tasks | O(n) |
| Delete Task | O(n) |

### Advantages of Linked Lists over Arrays

- Dynamic size.
- Easy insertion and deletion.
- No need to shift elements after insertion or deletion.
- Efficient memory utilization.

## Summary

A Singly Linked List is suitable for managing dynamic tasks where frequent insertion and deletion operations are required.
