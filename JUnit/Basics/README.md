# **JUNIT TESTING EXERCISES PROJECT**

This project demonstrates basic JUnit testing concepts using Java and Maven.

------------------------------------------------------------

# **EXERCISE 1: JUNIT SETUP**

This exercise focuses on setting up JUnit in a Maven-based Java project and verifying that the test environment is working correctly.

**CLASS:** SampleTest.java  
**PATH:** src/test/java/com/example/  
**GOAL:** Verify JUnit setup is working  
**OUTPUT:** Successful test execution

------------------------------------------------------------

# **EXERCISE 2: BASIC UNIT TESTING**

This exercise demonstrates writing basic unit tests for a simple Java class. It includes logic to check whether a number is Positive, Negative, or Zero.

**MAIN CLASS:** NumberUtil.java  
**TEST CLASS:** NumberUtilTest.java  

**PATHS:**  
src/main/java/com/example/  
src/test/java/com/example/  

**GOAL:** Learn basic unit testing using JUnit  
**OUTPUT:** All test cases passed successfully

------------------------------------------------------------

# **EXERCISE 3: ASSERTIONS IN JUNIT**

This exercise demonstrates the use of different JUnit assertions to validate expected results in test cases.

**CLASS:** AssertionsTest.java  
**PATH:** src/test/java/com/example/  
**GOAL:** Understand and use JUnit assertions  
**OUTPUT:** All assertions passed successfully

------------------------------------------------------------

# **EXERCISE 4: ARRANGE ACT ASSERT PATTERN AND SETUP TEARDOWN**

This exercise demonstrates the Arrange Act Assert pattern and the use of setup and teardown methods in JUnit testing.

**CLASS:** AAAExampleTest.java  
**PATH:** src/test/java/com/example/  

**GOAL:** Write structured test cases using AAA pattern  
**SETUP METHOD:** @Before  
**TEARDOWN METHOD:** @After  
**OUTPUT:** Clean and successful test execution
